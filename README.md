# vue-ant-watermark

## Project setup

```
npm install v-antd-watermark
```

### main.js

```
import WaterMark from 'v-antd-watermark';
Vue.use(WaterMark);

```

### use

```
<water-mark :content="'6666666'">
  <img alt="Vue logo" src="./assets/logo.png" />
  <HelloWorld msg="Welcome to Your Vue.js App" />
</water-mark>
```
